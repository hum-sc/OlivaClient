This document describes the process of deleting inside pages, this wit objectives to know when to delete a paga, when to move content from another page to the current page, and how to handle the layout when deleting pages.

## Deleting a page

When deleting inside a page, the following steps are taken:
1. Check if there is content in the current page.
    1. If there is no content, simply delete the page.
    2. If there is content
        1. Get paragraph nodes previous to the selection anchor.
        2. Get paragraph nodes after to the selection anchor.
